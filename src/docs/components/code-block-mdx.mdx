## Installation

Before creating the MDX component, make sure you have the basic component structure:

<DocCard folder="components" document="code-block" />

1. Install the following dependencies:

```bash
npm i @types/mdx -D
```

2. Create the Copy Button component:

<DocCard folder="components" document="copy-button" />

3. Create a `react-to-text` utility to get plain text from `pre` elements:

<ShowSource component="react-to-text" />

4. Finally, create the `<pre />` MDX component with our custom code block:

> `data-language` and `data-title` are added via custom [Shiki transformers](/docs/shiki/add-properties). You can delete if you don't need them.

<ShowSource component="code-block-mdx" />

## Usage

In your MDX Components object, add the new `PreMDXComponent` component:

```ts {3,6}
import type { MDXComponents as MDXComponentsType } from "mdx/types";

import { PreMDXComponent } from "@/components/code-block/pre-component";

const MDXComponents: MDXComponentsType = {
  ...PreMDXComponent,
};
```
