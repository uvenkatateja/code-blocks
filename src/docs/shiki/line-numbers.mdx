## Installation

1. Copy the `showLineNumbers` transformer into your project:

<ShowSource component="shiki-show-line-numbers" />

2. Add the following styles to your global CSS file to style the line numbers:

```css title="src/styles/globals.css"
pre.shiki-line-numbers code {
  counter-reset: step;
  counter-increment: step 0;
}

pre.shiki-line-numbers code:has(.line:nth-child(2)) .line::before {
  content: counter(step);
  counter-increment: step;
  width: 0.5rem;
  margin-right: 1rem;
  margin-left: 0.3rem;
  display: inline-block;
  text-align: right;
  @apply font-mono text-xs text-neutral-400 dark:text-neutral-600;
}
```

3. Import the transformer in your `rehypeShiki` plugin:

```ts title="Rehype Shiki Options"
import { showLineNumbers } from "@/utils/shiki/transformers/show-line-numbers";

const rehypeShikiOptions: RehypeShikiCoreOptions = {
  //...
  transformers: [showLineNumbers()],
};

export { rehypeShikiOptions };
```


## Usage

To enable line numbers in your code blocks, add the `lineNumbers` property to the code block in your MDX files:

````md title="MDX Code Block with Line Numbers" lineNumbers
```ts lineNumbers
````
