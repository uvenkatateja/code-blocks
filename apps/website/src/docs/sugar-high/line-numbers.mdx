## Usage

### Component

Add the `lineNumbers` prop to the [`CodeBlockSugarHigh`](/docs/react/code-block-client-sugar-high#usage) component:

```tsx title="Code Block with Line Numbers"
<CodeBlockSugarHigh lineNumbers={true} code={code} />
```

### Markdown/MDX

Add `sh-line-numbers` class to the [`PreSugarHighComponent`](/docs/react/code-block-mdx#highlighter-setup):

```tsx {2}
<CodeBlockContent>
  <pre className="sh-line-numbers">
    <code dangerouslySetInnerHTML={{ __html: codeHTML }} />
  </pre>
</CodeBlockContent>
```
